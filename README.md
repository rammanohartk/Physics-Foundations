# Physics Foundation
This repository focuses on building a structured foundation in mathematical, theoretical and computational physics.
The current stage include mathematical foundation. These  foundations are directly applied and one example is  the Euler solver implemented in the ODE-Solver project, where differential equations are solved computationally and validated against analytical solutions.
Further topics in quantum mechanics, classicalmechanics, electrodynamics and solid state  physics are planned as part of continued development.
